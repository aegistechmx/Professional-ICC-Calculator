Metodo de impedancias — Se calcula la impedancia equivalente del sistema desde la fuente hasta cada punto de interes.

Z_fuente = V / (sqrt(3) x Isc) [trifasico]

Z_trafo = (%Z/100) x V^2_sec / S_trafo

Z_alim = sqrt(R^2 + X^2) x L / 1000

Isc = V / (sqrt(3) x Z_total)

I_pico = Isc x sqrt(2) x (1 + e^(-pi*R/X))
